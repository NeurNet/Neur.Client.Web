# Neur.Client.Web

![banner](./.github/assets/banner.png)

## RU 🇷🇺

Веб-интерфейс для NeurNet

### Конфигурация

Перед сборкой и запуском проекта, отредактируйте файл `.env` и замените значение переменной `VITE_API_URL` на базовый эндпоинт запущенного [API NeurNet](https://github.com/NeurNet/Neur.Server.Net).

### Запуск в режиме разработки

```sh
$ npm i       # или `bun i`
$ npm run dev # или `bun run dev`
```

### Сборка

```sh
$ npm run build # или `bun run build`
```

## EN 🇺🇸

Web UI for NeurNet

### Configuration

Before the launch of the application, edit the `.env` file and set `VITE_API_URL` value to the base endpoint of the [NeurNet API](https://github.com/NeurNet/Neur.Server.Net).

### Run in development mode

```sh
$ npm i       # or `bun i`
$ npm run dev # or `bun run dev`
```

### Build

```sh
$ npm run build # or `bun run build`
```
